# Project Pitch

## 30-second Pitch

EventFlow Agent is a LangGraph-based Agentic Workflow system that simulates how a SaaS team triages external events such as service incidents, security advisories, API changes, and product releases.

It converts noisy public signals into **structured** Event Risk Briefs with evidence, dependency impact, risk assessment, and optional human review.
